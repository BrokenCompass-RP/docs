There are many controls available in the server. This page is an attempt to document all of the commonly used controls into a single place.

## Useful Controls

<Info>
  All controls assume default key-bindings. If you have changed your key-binds, these controls will likely be different.

  For a full list of key-binds, use the pause menu and navigate to Game \> Keybinds.
</Info>

## Useful Commands

All commands are entered into the chat window.

| Command | Description |
| --- | --- |
| /me (description) | Used in character to provide context to a scene. |
| /ooc | Used to deliver out of character messages. Should be used sparingly. |

## [Third-Eye](#third-eye)

Wherever you see or hear the concept of a 'third-eye', it's in relation to how you target objects in the world. To open your third eye, hold <kbd>alt</kbd> and move your camera to the direction of an object.

If an object is interactable via your third-eye, an icon will appear on the screen. Left clicking while this icon is visible will open the third-eye menu listing all the actions that can be performed.

Use your mouse to select the appropriate action and <kbd>left click</kbd> to perform it.

## [Radial Menu](#radial-menu)

There are various context related actions that can be performed inside of the radial menu. The radial menu is a pop-up UI element that when activated, can be used to cycle through these different context related actions.

To open the radial menu, hold the <kbd>Z</kbd> key, then use your mouse cursor and left mouse button to cycle through the various actions.

<Note>
  Some actions will not be available under certain conditions. For example, job related actions will not be visible if you are not on-duty or assigned to that job.
</Note>

## [Taxi Service](#taxi-service)

You may notice a taxi projection appear around various locations in the city. These are courtesy stops you can access to hail a cab to take you to other pre-determined taxi locations.

To use this service, walk into the taxi projection and press the <kbd>E</kbd> key to open the taxi service menu.

Select from a list of pre-defined locations, or select the option to take you to your GPS waypoint.

### Using the /taxi Service

A courtesy taxi is available to citizens of Los Santos in the event that you cannot hail a regular cab. Using the **/taxi** command in chat will call a taxi to your location.

<Note>
  Remember that roleplay should always come first. The **/taxi** service is available as a courtesy if you cannot find a taxi via RP means. Try to use it only when unable to engage in player to player taxi roleplay.
</Note>

#### Starting a Route

Open your map and double click the area you want to navigate to. A purple waypoint should appear.

Once the taxi arrives and you enter, enter the command **/taxigo** into the chat to begin your travel.

<Danger>
  **NOTE:** The **/taxigo** command requires you select your GPS waypoint before using it, or that you select a pre-determined destination before using it. If you do not, the taxi will fail and you'll need to use **/taxistop** and restart the route selection process.
</Danger>

#### Stopping a Route

Want to be dropped of early, or cancel your route? Enter the command **/taxistop** into the chat to cancel your current cab, or any incoming taxi service.

<Warning>
  Sometimes, the taxis in the city get lost. It is a big city after all. In the event that your taxi gets stuck or does not arrive after some time, use the **/taxistop** command to cancel the taxi. You can then call it again.
</Warning>
